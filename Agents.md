# Agent Notes

## Project Overview

- Project name: `todo-app`
- Purpose: a lightweight todo application with local persistence

## Technology Stack
- React 19
- TypeScript
- CSS by styled component

## Coding Standards & Patterns
- Enforce strict typing everywhere. No usage of `any`.
- Prefer explicit errors and return statements over silent fallbacks.
- Use PascalCase for React component files.

## Automation & Verification Tasks
- Build command: `npm run build`
